# TabulaRasa

**TabulaRasa** is a CSS stylesheet that aims to reset and normalize the default style of any modern browser.

## Usage

Include `TabulaRasa.css` or ``TabulaRasa.min.css`` in the head of your HTML file before your main stylesheet:

```html
...
<head>
    ...
    <link rel="stylesheet" href="TabulaRasa.css">
    <link rel="stylesheet" href="main.css">
    ...
</head>
...
```

Otherwise import either of these files at the beginning of your main `.css` or `.scss` stylesheet:

```css
@import url("TabulaRasa.css");
```
